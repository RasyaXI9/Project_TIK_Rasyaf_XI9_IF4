API UNGGUN

Api unggun saat kegiatan pramuka di SMAN 3 Bandung menjadi salah satu momen paling ditunggu-tunggu. Kegiatan ini biasanya diadakan di area terbuka, seperti lapangan sekolah atau lokasi camping di alam terbuka. Suasana malam yang tenang dan bintang-bintang yang bersinar menambah kehangatan kegiatan ini.


Selama acara, anggota pramuka berkumpul di sekitar api unggun untuk berbagai aktivitas. Mereka menyanyikan lagu pramuka, berbagi cerita, dan melakukan permainan yang mempererat rasa kebersamaan. Api unggun bukan hanya sekadar sumber cahaya dan kehangatan, tetapi juga menjadi tempat untuk membangun ikatan antar anggota. Setiap peserta berperan aktif, baik dalam menyiapkan api unggun maupun dalam merencanakan aktivitas malam. Selain itu, mereka juga diajarkan tentang keselamatan dan cara mengelola api dengan baik. Ini penting untuk memastikan semua orang tetap aman dan lingkungan terjaga. Momen di sekitar api unggun di SMAN 3 Bandung tidak hanya menjadi ajang bersenang-senang, tetapi juga mengajarkan nilai-nilai kebersamaan, tanggung jawab, dan cinta alam. Kenangan yang tercipta di sini akan selalu diingat sebagai bagian dari perjalanan pramuka yang berharga.
